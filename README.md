# 📝 Flutter TodoApp

A clean, efficient, and user-friendly To-Do application built with **Flutter**. This app is designed to help users manage their daily tasks, stay organized, and improve productivity with a modern mobile interface.

## 🚀 Features

* **Task Management**: Easily add, edit, and delete tasks.
* **Task Completion**: Mark tasks as completed with a single tap.
* **Clean UI**: Minimalist and intuitive design for a better user experience.
* **Persistent Storage**: Your tasks are saved locally so you never lose them.
* **Real-time Updates**: Instant UI updates when tasks are modified.

## 🛠️ Tech Stack

* **Framework**: [Flutter](https://flutter.dev/)
* **Language**: [Dart](https://dart.dev/)

## 📦 Installation & Setup

Follow these steps to run the project locally:

1. **Clone the repository:**
```bash
git clone https://github.com/eminegules1/Flutter-TodoApp.git

```


2. **Navigate to the project directory:**
```bash
cd Flutter-TodoApp

```


3. **Install dependencies:**
```bash
flutter pub get

```


4. **Run the application:**
```bash
flutter run

```
